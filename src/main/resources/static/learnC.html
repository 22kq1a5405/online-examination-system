📘 Introduction to C Language

C is a powerful, general-purpose programming language that was developed in the early 1970s by Dennis Ritchie at Bell Labs. It is often referred to as the "mother of all programming languages" because many modern languages like C++, Java, and Python are based on C.

📌 Key Characteristics of C:

  • ✅ Structured and procedural language
  • ✅ Fast and efficient
  • ✅ Offers low-level memory access using pointers
  • ✅ Ideal for system programming and embedded systems
  • ✅ Portable – C programs can run on different machines with minimal changes

💡 Where is C Used?

  • 💻 Operating system development (e.g., Unix, Linux)
  • 🔧 Embedded systems and firmware
  • 🧮 Compilers and interpreters
  • 🎮 Game development engines
  • 📱 Mobile and desktop applications (with system-level interaction)

C continues to be a popular language for both academic and industrial purposes. Learning C helps build a strong foundation in programming, as it teaches core concepts like memory management, data structures, and algorithm development.

📐 Structure of a C Program

A basic C program follows a defined structure. Understanding this structure helps in writing error-free and maintainable code. Below is the general syntax of a C program:

📄 General Structure:

#include <stdio.h>  // Preprocessor Directive

// Main Function
int main() {
    // Statement
    printf("Hello, World!");
    return 0;
}

💡 Explanation of Components:

  • #include <stdio.h> – Preprocessor directive that includes the Standard Input Output library for functions like printf() and scanf().
  • main() – Starting point of every C program. It’s where execution begins.
  • { } – Curly braces define the body of the function or code block.
  • printf() – A standard function used to display output on the screen.
  • return 0; – Ends the main function and returns control to the operating system.

📌 Key Points:

  • Each C statement ends with a semicolon ;
  • Code inside functions is enclosed in curly braces { }
  • Execution of every C program starts from the main() function
  • Whitespace and indentation improve readability but are not mandatory

🎯 Tip: Practice writing small programs using the basic structure to strengthen your understanding of C syntax.

🔤 Variables in C

In C programming, a variable is a container used to store data that can be modified during the execution of a program. Every variable in C must be declared with a data type before it is used.

📌 Syntax:

data_type variable_name;
int age;

🧠 Example:

#include <stdio.h>

int main() {
    int age = 20;
    float weight = 60.5;
    char grade = 'A';

    printf("Age: %d\n", age);
    printf("Weight: %.1f\n", weight);
    printf("Grade: %c\n", grade);
    return 0;
}

📚 Types of Variables:

  • int – stores integers (e.g., 5, -10)
  • float – stores decimal numbers (e.g., 3.14, -0.5)
  • char – stores single characters (e.g., 'A', 'b')

🛠️ Rules for Naming Variables:

  • Must begin with a letter (A-Z or a-z) or an underscore (_) but not a digit
  • Can contain letters, digits, and underscores
  • No special characters or spaces allowed
  • Case-sensitive (e.g., Age and age are different)

🔎 Note: Declaring the correct type of variable helps C manage memory efficiently and avoid errors during execution.

🔄 Type Casting in C Language

Type Casting in C refers to converting a variable from one data type to another. It is used when you want to perform operations between different data types or store one type of value into a different type of variable.

📌 Types of Type Casting:

  • 1. Implicit Type Casting (Automatic)
    • Performed by the compiler automatically.
    • Occurs when a smaller data type is converted into a larger data type.
    • Example: int to float
  • 2. Explicit Type Casting (Manual)
    • Performed by the programmer using the cast operator.
    • Syntax: (type) variable
    • Example: float to int

🧠 Example:

#include <stdio.h>

int main() {
    int a = 10;
    float b = 5.5;

    // Implicit type casting
    float result = a + b;

    // Explicit type casting
    int c = (int)b;

    printf("Result (float): %.2f\n", result);
    printf("Converted value (int): %d\n", c);

    return 0;
}

🔍 Output:

Result (float): 15.50
Converted value (int): 5

🎯 Note: Be careful while using type casting, as converting from a higher type to a lower type (like float to int) can lead to data loss.

Operators in C are special symbols used to perform operations on variables and values. They are essential for performing arithmetic, logical, relational, and other types of computations.

📋 Types of Operators:

🎯 Tip: Understanding operator precedence and associativity is crucial for writing correct expressions in C.

🔁 Control Statements in C

Control statements in C language determine the flow of program execution. They help make decisions, repeat operations, and control how and when specific blocks of code are executed.

1. Conditional Control Statements

  • if: Executes a block of code if a condition is true.
  • if-else: Executes one block if the condition is true and another block if false.
  • else if ladder: Checks multiple conditions one after another.
  • switch: Selects one block of code to run from many options.
int num = 10;
if(num > 0) {
    printf("Positive number");
} else {
    printf("Non-positive number");
}

2. Looping Statements

  • for: Used when the number of iterations is known.
  • while: Executes as long as the condition is true.
  • do-while: Executes at least once before checking the condition.
// for loop example
for(int i = 0; i < 5; i++) {
    printf("%d ", i);
}

3. Jump Statements

  • break: Exits from the loop or switch statement.
  • continue: Skips the current iteration and continues with the next.
  • goto: Jumps to a labeled statement.
  • return: Exits from the function and optionally returns a value.
// break example
for(int i = 0; i < 5; i++) {
    if(i == 3) break;
    printf("%d ", i);
}

✅ Conclusion: Control statements enhance the flexibility of C programs by directing the flow of execution based on various conditions.

🔧 Functions in C Language

A function in C is a block of code that performs a specific task. It helps in breaking down a large program into smaller, manageable pieces. Functions make code modular, reusable, and easier to understand and debug.

📌 Types of Functions:

  • Library Functions: Predefined functions provided by C (e.g., printf(), scanf()).
  • User-defined Functions: Functions created by the user to perform specific tasks.

🧱 Syntax of a User-defined Function:

return_type function_name(parameters) {
    // code block
    return value;
}

🧪 Example:

#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    printf("Sum: %d", result);
    return 0;
}

⚙️ Components of a Function:

  • Function Declaration: Tells the compiler about the function name and parameters.
  • Function Definition: Contains the actual code of the function.
  • Function Call: Invokes the function to execute.

💡 Advantages of Using Functions:

  • Improves code readability and modularity
  • Reduces code repetition
  • Makes testing and debugging easier
  • Encourages code reuse

🎯 Tip: Keep your functions focused on a single task and use meaningful names for better clarity.

📚 Arrays in C Language

An array in C is a collection of elements of the same data type stored in contiguous memory locations. It allows you to store multiple values in a single variable instead of declaring separate variables for each value.

🔢 Why Use Arrays?

  • To store multiple values of the same type efficiently
  • To reduce code complexity and redundancy
  • To access and manipulate data using loops easily

📌 Syntax:

data_type array_name[size];

Example:

int numbers[5] = {10, 20, 30, 40, 50};

🔍 Accessing Array Elements

Array elements are accessed using their index. Indexing starts from 0.

printf("%d", numbers[2]); // Outputs 30

🛠 Types of Arrays

  • One-Dimensional Array: Stores elements in a single row
  • Two-Dimensional Array: Stores elements in rows and columns (matrix form)
  • Multi-Dimensional Array: Arrays with more than two dimensions

🧪 Example: Traversing an Array

#include <stdio.h>

int main() {
    int i;
    int numbers[5] = {10, 20, 30, 40, 50};

    for(i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }
    return 0;
}

✅ Advantages:

  • Easy to manage large sets of data
  • Efficient memory access
  • Greatly simplifies looping structures

🎯 Tip: Always make sure to access array elements within the valid range (0 to size-1) to avoid memory errors.

📍 Pointers in C Language

A pointer in C is a variable that stores the memory address of another variable. Pointers are powerful tools that allow direct memory access, dynamic memory management, and efficient array and function handling.

📌 Syntax:

data_type *pointer_name;

Example:

int a = 10;
int *ptr = &a;

🧠 Key Pointer Operators

  • & (Address-of operator): Returns the address of a variable.
  • * (Dereference operator): Accesses the value stored at the pointer's address.

🔍 Example: Using Pointers

#include <stdio.h>

int main() {
    int x = 42;
    int *ptr = &x;

    printf("Value of x: %d\n", x);
    printf("Address of x: %p\n", &x);
    printf("Pointer ptr holds address: %p\n", ptr);
    printf("Value pointed to by ptr: %d\n", *ptr);

    return 0;
}

📚 Uses of Pointers

  • Efficient handling of arrays and strings
  • Dynamic memory allocation (using malloc(), calloc())
  • Passing large structures to functions efficiently
  • Creating data structures like linked lists and trees

✅ Advantages of Pointers

  • Improves performance by direct memory access
  • Supports dynamic memory and data structure manipulation
  • Makes function arguments more flexible (pass by reference)

🎯 Tip: Pointers are powerful but must be used carefully. Accessing invalid memory can cause segmentation faults.
