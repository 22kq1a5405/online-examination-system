ğŸ§­ Control Statements in Python

Control statements in Python manage the flow of a program based on conditions and logic. These include conditional statements, loops, and loop control statements.

1. Conditional Statements

  • if: Executes a block of code if a condition is true.
  • if-else: Executes one block if the condition is true, another if it is false.
  • if-elif-else: Checks multiple conditions in sequence.
age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

2. Looping Statements

  • for loop: Iterates over a sequence (like a list or range).
  • while loop: Repeats as long as a condition is true.
# For loop example
for i in range(3):
    print(i)

# While loop example
count = 0
while count < 3:
    print(count)
    count += 1

3. Loop Control Statements

  • break: Exits the loop prematurely.
  • continue: Skips the current iteration and moves to the next.
  • pass: Does nothing; a placeholder statement.
# break example
for i in range(5):
    if i == 3:
        break
    print(i)

# continue example
for i in range(5):
    if i == 3:
        continue
    print(i)

ğŸ“ Note: Proper use of control statements is key to writing efficient and readable code in Python.
