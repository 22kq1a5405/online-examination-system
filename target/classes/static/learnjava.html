Java vs Python

Java is a statically-typed, compiled language known for its speed, portability, and use in enterprise applications. Python, on the other hand, is a dynamically-typed, interpreted language popular for its simplicity and use in data science, scripting, and automation.

Java Input and Output (I/O)

In Java, Input and Output (I/O) is used to read data from the user and display information to the user. Java provides several ways to handle I/O operations. The most common are:

üîπ Output in Java

To display output on the screen, Java uses the System.out object and its println() and print() methods. 

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
  • System.out.print(): prints text without moving to a new line
  • System.out.println(): prints text and moves to the next line

üîπ Input in Java

For user input, Java commonly uses the Scanner class from the java.util package. 

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        System.out.print("Enter your name: ");
        String name = input.nextLine();
        System.out.println("Hello, " + name);
    }
}

üìå Common Scanner Methods

  • next() - reads a single word
  • nextLine() - reads a full line
  • nextInt() - reads an integer
  • nextDouble() - reads a double

üí° Notes

  • Always import java.util.Scanner to use it.
  • Use input.close() at the end to close the scanner (optional for small programs).

Java also supports advanced input/output using BufferedReader, PrintWriter, and file streams‚Äîbut for beginners, Scanner and System.out are enough to get started.

Java Variables

A variable in Java is like a container that holds data values during program execution. Each variable has a data type that defines the kind of data it can store.

For example: int age = 25;
Here, int is the data type, age is the variable name, and 25 is the value assigned.

Java is a statically-typed language, so you must declare the variable type before using it.

  • int ‚Äì for integers (e.g., 10, -5)
  • float ‚Äì for decimal values (e.g., 3.14f)
  • char ‚Äì for single characters (e.g., 'A')
  • String ‚Äì for text (e.g., "Java")
  • boolean ‚Äì for true/false values

Variables help make programs flexible by storing changing data.

Java Data Types

Java data types define the kind of data a variable can store. Java is a statically-typed language, which means variables must be declared with a type.

Types of Data Types

  • Primitive Data Types ‚Äì Basic built-in types like int, float, boolean, etc.
  • Non-Primitive Data Types ‚Äì Reference types like Strings, Arrays, Classes, etc.

Primitive Data Types

Java has 8 primitive data types:

  • int: 4 bytes ‚Äî used to store whole numbers.
    int age = 25;
  • float: 4 bytes ‚Äî used to store decimal numbers with single precision.
    float pi = 3.14f;
  • double: 8 bytes ‚Äî used for large decimal numbers with double precision.
    double price = 99.99;
  • char: 2 bytes ‚Äî used to store a single character.
    char grade = 'A';
  • boolean: 1 bit ‚Äî used for true or false values.
    boolean isJavaFun = true;
  • byte: 1 byte ‚Äî used for small numbers from -128 to 127.
    byte level = 100;
  • short: 2 bytes ‚Äî used for whole numbers from -32,768 to 32,767.
    short temp = 20000;
  • long: 8 bytes ‚Äî used to store large whole numbers.
    long population = 7896543210L;

Non-Primitive Data Types

Non-primitive data types refer to objects and include:

  • String ‚Äì Sequence of characters
    String name = "Java";
  • Array ‚Äì Collection of similar data types
    int[] numbers = {1, 2, 3};
  • Class ‚Äì Blueprint for creating objects
  • Interface ‚Äì Abstract type used to specify behavior

Java Control Statements

Java control statements are used to control the flow of execution in a program. These statements allow you to make decisions, repeat tasks, and branch to different parts of code.

Types of Control Statements in Java

  1. Decision-making Statements
    • if statement: Executes a block if the condition is true.
      if (a > b) { System.out.println("A is greater"); }
    • if-else statement: Executes one block if condition is true, otherwise another.
      if (a > b) { ... } else { ... }
    • else-if ladder: Checks multiple conditions.
      if (...) { ... } else if (...) { ... } else { ... }
    • switch statement: Selects one of many blocks to execute.
      switch(day) {
      case 1: System.out.println("Monday"); break;
      default: System.out.println("Other day");
      }
  2. Looping Statements
    • for loop: Repeats a block for a known number of times.
      for (int i = 0; i < 5; i++) { ... }
    • while loop: Repeats while condition is true.
      while (i < 5) { ... }
    • do-while loop: Executes at least once, then repeats while condition is true.
      do { ... } while (i < 5);
  3. Jump Statements
    • break: Exits a loop or switch block immediately.
      if (x == 5) break;
    • continue: Skips current iteration and moves to the next.
      if (x % 2 == 0) continue;
    • return: Exits from a method and optionally returns a value.
      return x + y;

These control statements help structure your Java programs to be efficient and logical.

Java Methods

A method in Java is a block of code that performs a specific task. It helps to organize code, improve reusability, and make programs easier to understand and maintain.

Syntax of a Method

returnType methodName(parameters) {
    // method body
}

Example: 

public int add(int a, int b) {
    return a + b;
}

Types of Methods in Java

  • Predefined Methods: Built-in methods provided by Java (e.g., System.out.println())
  • User-defined Methods: Methods created by the programmer to perform specific tasks

Method Parameters

Methods can take input values called parameters and return output using a return type.

You can also create void methods that do not return any value. 

public void greet() {
    System.out.println("Hello!");
}

Calling a Method

To execute a method, you need to "call" it: 

MyClass obj = new MyClass();
int result = obj.add(5, 10);

Method Overloading

Java allows multiple methods with the same name but different parameters. This is known as method overloading. 

public int multiply(int a, int b) {
    return a * b;
}

public double multiply(double a, double b) {
    return a * b;
}

Benefits of Using Methods

  • Code reusability
  • Better structure and readability
  • Easy debugging and maintenance

Java methods are a core part of object-oriented programming and essential for organizing code in classes and objects.

Java OOPs Concepts (Object-Oriented Programming)

Java is an object-oriented programming (OOP) language. OOP is a programming paradigm based on the concept of "objects", which contain both data and methods. OOP makes it easier to manage, reuse, and scale code.

Main Principles of OOP in Java

  1. 1. Class
    A class is a blueprint or template for creating objects. It defines the properties (fields) and behaviors (methods) that the object will have.
    class Car { int speed; void drive() { ... } }

  2. 2. Object
    An object is an instance of a class. It represents a real-world entity with state and behavior.
    Car myCar = new Car();

  3. 3. Inheritance
    Inheritance allows one class to acquire the properties and methods of another class. This promotes code reuse.
    class Dog extends Animal { ... }
    The class Dog inherits from Animal.

  4. 4. Polymorphism
    Polymorphism allows one interface to be used for different data types or objects. It means ‚Äúmany forms‚Äù. It is achieved using method overloading and method overriding.
    animal.sound(); // could be Dog or Cat

  5. 5. Abstraction
    Abstraction means hiding internal details and showing only essential features. This is done using abstract classes or interfaces.
    abstract class Shape { abstract void draw(); }

  6. 6. Encapsulation
    Encapsulation is the practice of keeping fields private and accessing them through public methods (getters/setters). It protects data from outside interference.
    class Person {
      private String name;
      public void setName(String n) { name = n; }
      public String getName() { return name; }
    }

Benefits of OOP

  • Improves code reusability using inheritance
  • Makes code more secure using encapsulation
  • Enhances flexibility and scalability through polymorphism
  • Increases maintainability by structuring code around real-world objects

In summary, Java‚Äôs object-oriented features make it powerful, modular, and suitable for large-scale applications.
